Angular Material Datepicker with Firestore Timestamps example

Firebase is changing how Dates are stored. Going forward, when a Date is part of an object that is saved to Firestore, Firestore will automatically convert it into a Timestamp. This will break Angular Material Datepickers if they are backed by Date objects. This project is an example of how to make Datepickers work with Firestore Timestamps.

To use this project you need to fill in your Firebase project config into environment.ts

References

  • Issue in question: https://github.com/firebase/firebase-js-sdk/issues/726
  • Angular Material Docs: https://material.angular.io/components/datepicker/overview
